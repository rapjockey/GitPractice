# what is REST-API -> api is application program interface which is used for connect the differents services like client and server, REST api is architecure based, used a json data i.e arrays of objects, api is a fuel for web apps.

# POSTMAN -> it is application used for testing api's where we can GET data and all http methods can perform and we can find a status codes, also we can create a collection of api's that can be share to other devlopers.

# JSON Server  -> created a json server using npm and fetch a api in js

# Idempotent -> used to avoid duplicate data, mostly used for POST method others are default idempotent, to avoid this types of error we create a uuid in frontend and server saved in cookies and when user retry the process the uuid matches and duplicate data will not save.

# Swagger UI -> it is like postman, it is used for testing apis , provides api documentation and collabrative interface for developers
